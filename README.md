# AWSKRUG #Women-In-Cloud 소모임

- since 2023


# 모임 소개

- 여성 엔지니어들은 조직에서 소수인 경우가 많지만, 서로의 존재를 알기 쉽지 않습니다.
- 조직 내에서 소수인 여성 엔지니어들이 서로를 알 수 있는 자리를 만들고, 지지해 줄 수 있는 네트워크를 만듦으로써,
- 여성 엔지니어들이 업계에서 성장하며 커리어를 오래 쌓을 수 있었으면 하는 마음으로 이 모임을 기획하게 되었습니다.
